# ⚡ Fast YouTube Downloader

O **Fast YT Downloader** é um CLI criado para baixar vídeos e áudios do YouTube de forma eficiente. A proposta do projeto é oferecer uma alternativa mais leve e rápida em comparação com soluções tradicionais baseadas em linguagens interpretadas, ja que a execução do projeto se resume a execução de um binário de Nim.

---

## Como rodar

Clone o projeto
```bash
git clone https://github.com/GuilhermeeDev/fast-yt-downloader.git && cd fast-yt-downloader
```

Build projeto com `Nimble`
```bash
nimble build
```
